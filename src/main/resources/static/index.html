W-MDSP API

A dominating set in a weighted graph is a set of vertices such that every vertex outside the set is adjacent to a vertex in the set, and the sum of edge weights incident to the set for every vertex is at least K. The domination number is the minimum cardinality of a dominating set in the graph. The problem of finding the minimum weighted dominating set is a combinatorial optimization problem that has been proved to be NP-hard. Given the difficulty of this problem, an Iterated Greedy algorithm is proposed for its solution and it is compared to the solution given by an exact algorithm and by the state-of-art algorithms.


Main Features

Our implementation includes the following Greedy algorithms:

  • Basic Greedy Algorithm: This algorithm utilizes insertion, destruction, and reconstruction phases. For each phase, we have defined four different heuristics.
  • Bioinspired Greedy Algorithm: This algorithm allows for the selection of constructive and destructive heuristics using a Wheel selection mechanism.

We have also incorporated an exact resolution method; however, its capabilities are limited. Also, our Java API is designed to be easily extendable, allowing anyone to modify the code and make change requests in order to improve the implementation.
